# Operating System

## Course Details
- Type of Credit: Required
- Credits: 3
- Course Department: Computer Science/B/3
- Instructor: Prof. 廖峻鋒
- Prerequisites: N/A
- Session: fri09-12

## Course Description
本課程主要目的為介紹電腦作業系統的核心知識、其內部的系統模組架構、和整體運作機制。因此在修讀完此科目後，可以對作業系統整體結構與運行方式有完整的概念，在開發應用程式時，可運用這些基本知識讓系統更加高效、安全與穩定。
