# Digital System Lab

## Course Details
- Type of Credit: Required
- Credits: 0
- Course Department: Computer Science/B/2
- Instructor: Prof. 廖文宏
- Prerequisites: N/A
- Session: tue16-18

## Course Description
本課程為搭配數位系統導論之實驗部分，藉由兩周一次的實驗，學習數位電路與系統的設計方法，透過實作讓學生接觸並操作電子電路邏輯元件之接線、偵錯、除錯與功能驗證之流程。
