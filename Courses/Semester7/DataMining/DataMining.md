# Data Mining

## Course Details
- Type of Credit: Elective
- Credits: 3
- Course Department: Computer Science/M/1
- Instructor: Prof. 沈錳坤
- Prerequisites: N/A
- Session: thu09-12

## Course Description
本課程主要在培養同學資料探勘(Data Mining)技術與演算法之基礎。在授課方向上，將以培養同學資料探勘演算法的基礎研究能力為主。
先修課程：程式設計、線性代數、資料結構、演算法、資料庫系統

先備技術： Python, DBMS, Github
